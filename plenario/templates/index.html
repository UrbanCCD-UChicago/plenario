Plenar.io: A spatio-temporal open data platform

Over the past few years, all levels of government from the federal administration to individual municipalities have been moving toward an open data model, publicly releasing data on how government entities function. Plenar.io tells a unified story of place by centralizing data from these various sources in order to increase accessibility and help ordinary citizens and researchers draw connections between datasets. Data can be viewed via a map interface and accessed by download or via a RESTful API.

Applications (under development)

  • API documentation
  • Data Explorer
  • Grid Aggregation
